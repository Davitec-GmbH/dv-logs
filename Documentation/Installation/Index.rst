============
Installation
============

.. contents:: On this page
   :local:
   :depth: 2

Requirements
============

- PHP 8.2+
- TYPO3 v12.4 LTS, v13.4 LTS, or v14

Installation via Composer
=========================

.. code-block:: bash

   composer require davitec/dv-logs

The extension is activated automatically in Composer-based installations.

Verification
============

After installation, log in as an admin user and navigate to
:guilabel:`System > DV Logs`. The module should appear in the left sidebar.
